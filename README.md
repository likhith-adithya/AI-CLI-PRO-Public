# AI-CLI-HUB Public Issue Tracker

> A community-driven issue tracker and feature request board for [AI-CLI-HUB](https://github.com/likhith-adithya/AI-CLI-HUB) - Your AI Agents: One click and one command away.

---

## 📋 About This Repository

This is the **public issue tracker** for the AI-CLI-HUB VS Code extension. This repository is exclusively for:

- 🐛 **Bug Reports** - Report issues and unexpected behavior
- ✨ **Feature Requests** - Suggest new features or improvements
- 📚 **Documentation Feedback** - Help improve our docs

> **Note**: The actual source code is maintained in a private repository for security and development purposes.

---

## 🚀 Quick Start

### Report a Bug

1. Check if your issue already exists by [searching the issues](../../issues)
2. Click **"New Issue"** → **"Bug Report"**
3. Fill in all the required information
4. Submit and wait for our team's response

### Request a Feature

1. Check if the feature has already been requested by [searching the issues](../../issues)
2. Click **"New Issue"** → **"Feature Request"**
3. Describe your use case and desired functionality
4. Submit and engage with the community

---

## 📝 Guidelines for Creating Issues

### ✅ Do's

- ✓ Search existing issues before opening a new one
- ✓ Use the provided issue templates
- ✓ Be clear and descriptive in your title
- ✓ Include relevant environment details (OS, VS Code version, extension version)
- ✓ Provide screenshots or error logs when applicable
- ✓ Be respectful and constructive
- ✓ Follow up on your issues if asked for more information

### ❌ Don'ts

- ✗ Create duplicate issues
- ✗ Post off-topic or promotional content
- ✗ Include sensitive information (API keys, passwords, personal data)
- ✗ Vent frustration without constructive feedback
- ✗ Close/delete your issue after the team has started working on it
- ✗ Spam with multiple similar issues

## 🏷️ Issue Labels

We use labels to organize and prioritize issues:

| Label | Description |
|-------|-------------|
| `bug` | Something is broken or not working as expected |
| `feature-request` | Request for a new feature or functionality |
| `enhancement` | Improvement to existing features |
| `documentation` | Documentation improvements or clarifications |
| `high-priority` | Urgent issue requiring immediate attention |
| `medium-priority` | Important but not urgent |
| `low-priority` | Can wait for future releases |
| `in-progress` | Currently being worked on |
| `awaiting-user-feedback` | Need more information from the reporter |
| `won't-fix` | Decided not to implement |
| `duplicate` | Duplicate of another issue |
| `good-first-issue` | Great for newcomers to contribute |
| `help-wanted` | Looking for community contributions |

For a complete list, see [LABELS.md](LABELS.md)

---

## ⏱️ Response & Resolution Timeline

- **Acknowledgment**: We'll respond to new issues within 2-3 business days
- **High Priority**: Resolution targeted within 1-2 weeks
- **Medium Priority**: Resolution targeted within 2-4 weeks
- **Low Priority**: Will be reviewed and prioritized in future releases
- **Feature Requests**: May require community discussion before implementation

## 💬 Community Standards

### Be Respectful
We're all volunteers working together to make AI-CLI-HUB better. Treat everyone with respect.

### Be Clear
Provide enough context for us to understand and reproduce your issue.

### Be Patient
Our team reviews issues regularly, but may not respond immediately. We appreciate your patience.

### Be Constructive
Focus on solutions rather than blame. Suggest improvements rather than just pointing out problems.
