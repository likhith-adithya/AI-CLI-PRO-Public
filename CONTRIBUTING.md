# Joining the AI-CLI-PRO Journey 🚀

First off, thank you for being here! Whether you're reporting a bug, suggesting a feature, or just looking to help out, you're a vital part of what makes **AI-CLI-PRO** great.

This repository is our **community hub**. While the core engine is developed in a private repository for security and performance reasons, our roadmap is driven entirely by **you**.

---

## 🗺️ How You Can Contribute

### 1. 🐛 Reporting Bugs
Spotted something that doesn't feel right? Help us squash it!
*   **Search First:** Check if someone else has already reported the issue.
*   **Be Specific:** Use our [Bug Report Template](../../issues/new?template=bug_report.md) to provide as much context as possible (OS, IDE version, steps to reproduce).

### 2. ✨ Requesting Features
Have an "Aha!" moment? We want to hear it!
*   **Dream Big:** No idea is too small or too ambitious.
*   **Show the Value:** Explain how the feature would improve your workflow. Use our [Feature Request Template](../../issues/new?template=feature_request.md).

### 3. 📚 Improving Documentation
Docs are the lifeblood of any project. If you find a typo or a confusing section, let us know!

---

## 📝 Our Shared Standards

To keep our community healthy and productive, we ask all contributors to:
*   **Be Respectful:** We follow a strict [Code of Conduct](CODE_OF_CONDUCT.md).
*   **Be Clear:** The more detail you provide, the faster we can help.
*   **Be Patient:** We're a small team, but we read every single issue.
